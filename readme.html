description:

run server:
1. npm install
2. windows: node bin\www
linux\os: node bin/www

This project runs a server that can give a login and registration pages.
if the user is logged in the server sends a weather page.
the server also manages the data base and serves a rest api for the user to operate on the data base.

Notes:

data base operations
We made a protected rest api that gives the user the option to operate his data base.
the api will send 400 error if the user isn't logged in.

data base structure
There is two tables in our data base, first a users table which contains all users info.
the second table is a places table which contains all places that users saved, with the id of the user that saved the place.

migrations file
We used the sequelize-mig library in order to create the migration file based on the model files.
